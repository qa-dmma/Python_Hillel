ДЗ 1.1. Підготовка робочого місця

    Завантажити та встановити Python (Не забуваємо про встановлення галочки "Add python to PATH")
    Перевірити правильність установки і доступність python. Для цього в консолі (терміналі) вводимо команду:
    python -V (у відповідь повинні побачити Python 3.x.x - де х.х будуть цифри,
    що вказують на версію встановленого інтерпретатора)
    Завантажити та встановити середовище розробки (IDE) PyCharm
    Створити перший проект.
    Додати до нього новий python файл
    Набрати в цьому файлі першу програму: print('Hello World!')
    Запустити програму на виконання
    Переконатися, що програма виконалася без помилок

Надати як рішення ДЗ скрини пунктів 2 і 7
